Labels

Description
===========
By customizing the trace command of the MTASC compiler you now will be able to route 
messages directly from your ActionScript classes to the well known Firebug console.

Usage
=====
Simply add -trace ThunderBolt.trace to the MTASC command line arguments.


Sample debug information
=================
description		my.package.MyClass::myMethod[123] : type @ 18:59:59
file 			path/to/my/class/file.as
fullClass 		my.package.MyClass
line 			123
method 			myMethod
time 			18:59:59
type 			string
Log 			levels


Debug Levels
============
By passing an extra literal to your trace actions, you can achieve multiple debug levels.

d 	debug 		trace('d This is a simple debug message.');
i 	info 		trace('i An info icon will appear next to this line.');
w 	warning		trace('w This message is highlighted.');
e 	error 		trace('e Red and highlited messages.)';
